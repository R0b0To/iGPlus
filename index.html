iGPlus

iGPlus is a browser extension for iGP Manager, available for Chrome and Firefox.

What iGPlus Does

iGPlus enhances the iGP Manager game interface by injecting additional HTML elements directly into the website. These enhancements provide:

  • Quick-access shortcuts to frequently used features
  • Additional race and strategy information
  • Additional report information
  • Improved usability and interface efficiency
  • Quality-of-life improvements for daily gameplay

All modifications are applied locally in your browser and do not alter the original game servers or backend systems.

Data Usage & Privacy

iGPlus does not collect, store, or transmit personal user data to any external servers controlled by the developer.

All extension data (such as settings and preferences) is stored locally within your browser.

Google OAuth & Optional Features

iGPlus optionally integrates with Google services using OAuth 2.0 for the following features:

  • Cloud sync of extension settings via Google Drive
  • Exporting race results to a user-selected Google Spreadsheet

These features are entirely optional and only when the user explicitly chooses to connect their Google account.

When enabled:

  • Data is stored directly in the user’s own Google Drive
  • No data is stored on external developer servers
  • Access is limited strictly to files created or selected by the user

Transparency & Permissions

The extension requests only the permissions necessary to:

  • Modify the iGP Manager webpage interface
  • Store local extension settings
  • Access Google Drive and Sheets (only if the user enables cloud features)

No background data collection or tracking is performed.
