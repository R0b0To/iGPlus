# Privacy Policy for iGPlus

**Effective Date:** May 08, 2026

iGPlus is a browser extension designed to enhance the user experience on igpmanager.com. This Privacy Policy explains how data is handled when using the extension.

---

## 1. Data Collection and Usage

iGPlus is a **local-first extension**.

* **No Personal Data Collection:**
  iGPlus does not collect, store, or transmit personal data to any external servers controlled by the developer.

* **Local Processing:**
  All functionality (including interface enhancements, race strategy tools, and report generation) is executed locally within the user’s browser.

* **Local Storage:**
  Extension settings and data are stored locally using browser storage APIs.

---

## 2. Dropbox Integration (Optional Feature)

iGPlus provides optional cloud synchronization features using Dropbox OAuth 2.0. These features are only activated when the user explicitly connects their Dropbox account.

### Data Accessed

When Dropbox integration is enabled, iGPlus accesses the following Dropbox user data:

* Dropbox account email address
* Basic profile information provided by Dropbox (such as account name)
* Files and folders created and managed by iGPlus within the user’s Dropbox account
* OAuth access tokens (used only to authenticate and maintain the session with Dropbox APIs)

### Data Usage

Dropbox user data is used strictly for the following purposes:

* Email address and profile information: to identify and authenticate the user within the extension
* Dropbox storage data: to store and retrieve user-generated configuration files, strategies, backups, and race reports
* OAuth tokens: to securely maintain access to Dropbox services during the active session

iGPlus does not use Dropbox user data for advertising, profiling, analytics, or any unrelated purpose.

### Data Storage

* All synchronized data remains entirely under the user’s Dropbox account.
* The extension does not store Dropbox user data on any external servers controlled by the developer.
* Local browser storage is only used for non-sensitive settings and preferences.

### Data Sharing

iGPlus does not sell, share, or transfer Dropbox user data to any third parties.

### Data Retention and Deletion

Users retain full control over their data at all times.

Dropbox data can be deleted by:

* Removing files or folders created by iGPlus in Dropbox
* Disconnecting Dropbox integration within the extension
* Revoking access for iGPlus via the user’s Dropbox connected apps settings

Once access is revoked, the extension can no longer access Dropbox user data.

### Security

All communication with Dropbox APIs is performed using secure OAuth 2.0 authorization over encrypted connections. No passwords are ever accessed or stored by iGPlus.

---

## 3. Data Sharing

* iGPlus does **not sell, share, or transfer** user data to third parties
* Data remains:

  * In the user’s browser, or
  * In the user’s Dropbox account (if sync is enabled)

---

## 4. Permissions Usage

iGPlus only requests permissions necessary to:

* Modify the igpmanager.com interface
* Store local extension data
* Access Dropbox storage (only if the user enables cloud features)

These permissions are used solely to provide the described functionality.

---

## 5. Security

* iGPlus uses standard browser APIs and official Dropbox OAuth 2.0 mechanisms
* No sensitive data is transmitted to developer-controlled servers
* All communication with Dropbox services is performed over secure channels
