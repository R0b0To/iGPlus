# Privacy Policy for iGPlus

**Effective Date:** May 03, 2026

iGPlus is a browser extension designed to enhance the user experience on igpmanager.com. This Privacy Policy explains how data is handled when using the extension.

---

## 1. Data Collection and Usage

iGPlus is a **local-first extension**.

- **No Personal Data Collection:**  
  iGPlus does not collect, store, or transmit personal data to any external servers controlled by the developer.

- **Local Processing:**  
  All functionality (including interface enhancements, race strategy tools, and report generation) is executed locally within the user’s browser.

- **Local Storage:**  
  Extension settings and data are stored locally using browser storage APIs.

---

## 2. Google Drive and Google Sheets Integration (Optional Feature)

iGPlus provides optional cloud features using Google OAuth 2.0. These features are only activated when the user explicitly connects their Google account.

### Data Accessed

When Google integration is enabled, iGPlus accesses the following Google user data:

- Google account email address
- Basic profile information provided by Google (such as name and profile picture)
- Google Drive files created and managed by iGPlus within a dedicated folder named `iGPlus`
- Google Sheets explicitly selected by the user for import or export functionality
- OAuth access tokens (used only to authenticate and maintain the session with Google APIs)

### Data Usage

Google user data is used strictly for the following purposes:

- Email address and profile information: to identify and authenticate the user within the extension
- Google Drive data: to store and retrieve user-generated configuration files, strategies, and race reports
- Google Sheets data: to allow users to export and import race-related data
- OAuth tokens: to securely maintain access to Google services during the active session

iGPlus does not use Google user data for advertising, profiling, analytics, or any unrelated purpose.

### Data Storage

- All data stored in Google Drive or Google Sheets remains entirely under the user’s Google account.
- The extension does not store Google user data on any external servers controlled by the developer.
- Local browser storage is only used for non-sensitive settings and preferences.

### Data Sharing

iGPlus does not sell, share, or transfer Google user data to any third parties.

### Data Retention and Deletion

Users retain full control over their data at all times.

Google Drive and Google Sheets data can be deleted by:
- Removing files or folders created by iGPlus in Google Drive
- Deleting or modifying connected Google Sheets
- Revoking access for iGPlus via the user's Google Account permissions page

Once access is revoked, the extension can no longer access Google user data.

### Security

All communication with Google APIs is performed using secure OAuth 2.0 authorization over encrypted connections. No passwords are ever accessed or stored by iGPlus.

---

## 3. Data Sharing

- iGPlus does **not sell, share, or transfer** user data to third parties
- Data remains:
  - In the user’s browser, or
  - In the user’s Google Drive (if sync is enabled)

---

## 4. Permissions Usage

iGPlus only requests permissions necessary to:
- Modify the igpmanager.com interface
- Store local extension data
- Access Google Drive and Google Sheets (only if the user enables cloud features)

These permissions are used solely to provide the described functionality.

---

## 5. Security

- iGPlus uses standard browser APIs and official Google OAuth 2.0 mechanisms
- No sensitive data is transmitted to developer-controlled servers
- All communication with Google services is performed over secure channels
