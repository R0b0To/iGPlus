# Privacy Policy for iGPlus

**Effective Date:** May 03, 2026

iGPlus is a browser extension designed to enhance the user experience on igpmanager.com. This Privacy Policy explains how data is handled when using the extension.

---

## 1. Data Collection and Usage

iGPlus is a **local-first extension**.

- **No Personal Data Collection:**  
  iGPlus does not collect, store, or transmit personal data to any external servers controlled by the developer.

- **Local Processing:**  
  All functionality (including interface enhancements, race strategy tools, and report generation) is executed locally within the user’s browser.

- **Local Storage:**  
  Extension settings and data are stored locally using browser storage APIs.

---

## 2. Google Drive Integration (Optional Feature)

iGPlus provides optional cloud features using Google OAuth 2.0. These features are **only فعال when the user explicitly connects their Google account**.

### Authentication
- Secure authentication is handled via Google OAuth 2.0.
- iGPlus does not access or store your Google account password.

### Files Created and Managed
When enabled, iGPlus creates and manages a folder named:

**`iGPlus` (in the user’s Google Drive)**

Inside this folder, the extension may create and update:
- `config.json` — stores extension settings
- `strategies.json` — stores user-defined race strategies
- `reports.json` — stores race reports and exported data

### File Access Scope
- The extension only reads and writes files that it creates or that the user explicitly selects.
- No access is performed outside this scope.

### Google Sheets Integration
If the user enables spreadsheet features:
- The extension can read from and write to **Google Sheets selected by the user**
- This includes spreadsheets that are:
  - Owned by the user
  - Shared with the user
  - Publicly accessible (“anyone with the link”)

- Data written to spreadsheets is strictly related to:
  - Race results
  - Reports generated by the extension

### Data Control
- All synced data remains in the user’s Google Drive
- The developer has **no access** to this data at any time

---

## 3. Data Sharing

- iGPlus does **not sell, share, or transfer** user data to third parties
- Data remains:
  - In the user’s browser, or
  - In the user’s Google Drive (if sync is enabled)

---

## 4. Permissions Usage

iGPlus only requests permissions necessary to:
- Modify the igpmanager.com interface
- Store local extension data
- Access Google Drive and Google Sheets (only if the user enables cloud features)

These permissions are used solely to provide the described functionality.

---

## 5. Security

- iGPlus uses standard browser APIs and official Google OAuth 2.0 mechanisms
- No sensitive data is transmitted to developer-controlled servers
- All communication with Google services is performed over secure channels
